從 Scratch 到 Python

Python 是一種簡單易學但功能強大的程式語言，
被廣泛應用於遊戲開發、網站建設、人工智慧等領域。
本課程將帶領學生從圖形化的 Scratch 進階到文字型的 Python 程式設計，
開啟程式設計的全新世界！

Python 的應用領域

人工智慧：Python 是 AI 開發的首選語言

網頁開發：使用 Django 或 Flask 架構建立網站

數據分析：透過 Pandas 和 NumPy 處理複雜數據

遊戲開發：使用 Pygame 製作互動式遊戲
